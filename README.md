# 📚 Matemática Discreta — Notas de Estudo

Notas de Matemática Discreta escritas de estudante para estudante.
O objetivo é traduzir as definições formais do livro em uma linguagem mais próxima de quem está aprendendo, com exemplos práticos e conexões diretas com programação.

---

## 📖 Conteúdo

- **Capítulo 2 — Teoria dos Números:** Divisibilidade, MDC/MMC, Números Primos, Aritmética Modular, Equações Diofantinas, Teorema de Fermat e Função Phi de Euler.
- **Capítulo 3 — Relações de Recorrência:** Classificação, Solução em Forma Fechada, Métodos Telescópicos, Equação Característica e Princípio da Superposição.
- **Capítulo 4 — Princípios de Contagem:** PFC, Inclusão-Exclusão, Casa dos Pombos, Permutações e Combinações.
- **Capítulo 6 — Teoria dos Grafos:** Definições, Terminologia, Planaridade, Isomorfismo, Árvores e Representação Computacional.

---

## 🛠️ Como usar

O material foi feito no **[Obsidian](https://obsidian.md/)** e usa recursos nativos da ferramenta: links internos, callouts e um CSS customizado para destacar elementos das equações.

Para a melhor experiência:
1. Clone ou baixe o repositório.
2. Abra a pasta como um vault no Obsidian.
3. Ative o snippet CSS em **Configurações → Aparência → CSS Snippets**.

Quem preferir, pode ler os arquivos `.md` diretamente pelo GitHub — o conteúdo é acessível, só as imagens internas e o estilo visual das equações não renderizam.
